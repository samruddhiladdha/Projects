 1. Quiz App | Python, Json (GitHub)
    - To start a quiz, use the requests library to generate a random link from the Trivia API.
    - The quiz begins with an initial score of zero and displays the first question.
    - As the quiz progresses, each answer is evaluated, and the next question is displayed, along with the updated score.

 2. Cartoonify Image | Python, OpenCV (GitHub)
    - To create a beautiful cartoon image from a photograph, start by converting the image to grayscale.
    - Next, smooth the grayscale image and extract the edges.
    - Finally, combine the edges with the original color image, masking the color with the edges to achieve a lighted, cartoon-like effect.
